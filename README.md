# Overview
This project analyzes social media behavior among Gen-Z users using a large-scale dataset of 1,000,000 records. The analysis spans five core areas: usage pattern exploration, mental health relationship studies, addiction-level classification, user segmentation via clustering, and cross-country behavioral comparisons.

# Dataset
Source    :  https://www.kaggle.com/datasets/sharmajicoder/gen-z-social-media-usage-dataset

# Features
* **Age** : User age
* **Gender** : User gender
* **country** : 7 countries (India, USA, Canada, UK, Germany, Brazil, Australia)
* **daily_usage_hours** : Average daily social media usage in hours
* **primary_platform** : Instagram / YouTube / TikTok / Twitter / Snapchat
* **num_platforms_used** : Number of platforms actively used
* **purposes** : Entertainment / Socializing / Education / News / Content Creation
* **avg_session_minutes** : Average time per session in minutes
* **night_usage** : Whether user uses social media at night (0/1)
* **mental_health_score** : Mental health score (higher = better)
* **addiction_level** : Low / Medium / High
* **screen_time_before_sleep** : Screen time before sleep in minutes

# Project Goals
1. Analyze social media usage patterns among Gen-Z <br>
2. Study the relationship between screen time and mental health <br>
3. Build models for addiction detection / classification <br>
4. Perform user segmentation and clustering <br>
5. Explore cross-country behavioral differences
